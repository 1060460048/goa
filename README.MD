## 介绍

基于 Beego + Vue 开发的在线问答系统。

## 功能

+ [x] 邮箱注册
+ [x] 邮件密码找回
+ [x] 会员邮件激活
+ [x] markdown内容提问和回答
+ [x] XSS安全过滤
+ [x] Vue前端小组件
+ [x] 完善的会员功能体系
+ [x] 后台前后端分离
+ [x] API接口
+ [x] 程序/数据库定时备份
+ [x] ENV环境配置
+ [ ] 单元测试
